Chrome Web Store 掲載情報（コピペ用）
============================================================

【ストアの掲載情報】タブ
------------------------------------------------------------

■ 名前（30文字以内）
Clean Start

■ 概要（132文字以内）
履歴・キャッシュ・Cookie・サイト保存データを 12 種から選んでワンクリックで一掃。期間指定と 2 段階確認で安心して使える、軽量・無料の Chrome 拡張機能。

■ 説明（16,000文字以内）
Chrome の閲覧データをすばやく整理するための、シンプルで安全なクリーンアップ拡張機能です。
ツールバーのアイコンをクリックして、削除したい項目と期間を選び、ボタンを 2 回タップするだけで完了します。

■ 主な機能

・ワンクリックの一括削除
  ポップアップから現在の設定でまとめてクリーンアップ。複雑な設定画面を行き来する必要はありません。

・12 種類のデータから選択
  履歴、キャッシュ、Cache Storage、Cookie、ダウンロード履歴、ファイルシステム、フォームデータ、IndexedDB、ローカルストレージ、Service Workers、Web SQL、App Cache のなかから自由に組み合わせ可能。

・期間指定
  「1時間以内」「24時間以内」「1週間以内」「4週間以内」「すべて」の 5 段階から削除対象期間を選べます。

・2段階タップで誤操作防止
  Clear ボタンは初回タップで「もう一度タップで削除」状態に切り替わり、5 秒以内に 2 回目をタップすると実行されます。誤クリックでセッションが消えるリスクを大幅に低減。

・Chrome 起動時の自動削除
  「Chrome 起動時に削除する」を有効にすると、ブラウザを起動するたびに自動でクリーンアップを実行。設定画面には destructive 性を伝える警告バナーを併記。

・起動時タブの自動リロード
  起動時自動削除でキャッシュ系・Cookie 系を消した場合、開いていた通常の Web タブをバッチで段階的にリロードして、消去後の最新状態に自動で更新。

・削除後のアクティブタブ自動再読み込み（任意）
  「削除後にアクティブタブを再読み込み」を有効にすると、Clear 押下後に開いているタブを自動でリロード。

・ローカル完結
  すべての処理はブラウザ内で完結。ネットワーク送信・外部 API 連携・トラッキング・テレメトリは一切ありません。

■ 使い方

1. Chrome Web Store からインストール
2. ツールバーの Clean Start アイコンをクリック
3. 「削除する項目」のチェックボックスで対象データ型を選ぶ
4. 「対象期間」のラジオボタンで時間範囲を選ぶ
5. Clear ボタンをタップ → 「もう一度タップで削除」が出たら 5 秒以内にもう一度タップ
6. 完了！

オプションページ（chrome://extensions の「拡張機能のオプション」または右クリック → オプション）からも同じ設定を変更でき、より広い画面で詳細な説明を確認できます。

■ カテゴリ
仕事効率化

■ 言語
日本語、English

【プライバシーへの取り組み】タブ
------------------------------------------------------------

■ このアイテムに単一の目的はありますか？
はい

■ 単一の目的の説明
ユーザーが選択した期間とデータ型に基づいて、Chrome の履歴・キャッシュ・Cookie・サイト保存データを一括削除する。

■ データの使用に関する開示

  ・個人を特定できる情報を収集しますか？
    いいえ

  ・ユーザーのデータを取り扱いますか？
    いいえ（ローカルの設定値以外はブラウザの外に出さない）

  ・ユーザーデータをどのように扱いますか？
    - ユーザーデータの販売: しない
    - ユーザーデータの本来の目的以外への使用や転送: しない
    - 信用力の判定や貸付目的での使用や転送: しない

■ 権限の正当性

  storage:
  ユーザーが選択した削除対象（dataToRemove）、期間（timePeriod）、自動再読み込みフラグ（autorefresh）、Chrome 起動時自動削除フラグ（clearonstartup）を chrome.storage.local にローカル保存するために使用します。データは外部に送信されません。

  browsingData:
  本拡張機能の中核機能です。chrome.browsingData.remove() を呼び出して、ユーザーが指定した期間とデータ型（履歴、キャッシュ、Cookie、IndexedDB、Service Workers など）をまとめて削除するために使用します。

  activeTab:
  ユーザーがオプションで「削除後にアクティブタブを再読み込みする」を有効にした場合、Clear ボタン押下直後にアクティブタブを reload するために使用します。バックグラウンドで他タブを盗み見るためには使用しません。

■ 外部サーバーとの通信について
本拡張機能は外部サーバーと一切通信しません。すべての処理は chrome.browsingData / chrome.cookies / chrome.tabs などのブラウザ標準 API を介してローカルで完結します。

■ リモートコードを使用していますか？
いいえ
（すべてのコードは拡張機能パッケージ内に含まれており、外部からのコード読み込みは一切行いません。Content Security Policy で `script-src 'self'; object-src 'self'` を明示しています。）
