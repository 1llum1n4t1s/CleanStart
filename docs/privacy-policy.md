# プライバシーポリシー - Clean Start

最終更新日: 2026年4月20日

## 概要

`Clean Start` は、Chrome 内の閲覧データをユーザー操作で削除するためのローカル拡張機能です。

## 収集する情報

この拡張機能は、個人情報、閲覧内容、認証情報を外部へ送信しません。

## 使用する権限

### `storage`
オプション設定をブラウザ内に保存するために使用します。

### `browsingData`
履歴、キャッシュ、Cookie、サイト保存データを Chrome の `chrome.browsingData.remove` API 経由で削除するために使用します。Cookie もこの権限の範囲内で一括削除されます。

### `activeTab`
ユーザーがポップアップから削除を実行したあと、必要に応じてアクティブタブを再読み込みするために使用します。

## 外部通信

拡張機能の削除処理や設定処理はすべてブラウザ内で完結します。リモートコードの読み込みは行いません。
